VARIABLE GLOBALES Y LOCALES

En cualquier lugar de la página la variable que tiene validez es la Global menos en el ámbito de la variable local que es ésta la que tiene validez.

<script>
var variableGlobal;
</script>


<script>
function miFuncion (){
var variableLocal;
};
</script>


En la medida de lo posible y cómo consejo: No utilizar el mismo nombre de variable para variables globales y variable locales.

<script>
var num = 2;
function miFuncion (){
 var num = 10;
 document.write(num); // Imprime 10.
}
document.write(num); // Imprime 2.
</script>

Sino utilizamos la palabra var para declarar variables se trataran como variables globales aunque estén en un ambito local, por tanto se corre el riesgo de sobreescribir el valor de la variable global.

var num = 2;
function miFuncion (){
 num = 10;
 document.write(num); // Imprime 10;
}
document.write(num); // Imprime 2
//Llamamos a la funcion
miFuncion();
document.write(num); // Imprime 10;

 Sentencias si la expresión tiene como valor valor1
